# 一、拓展 1：非对称网络效应下的最优补贴方向

这一部分主要分析：

[
\alpha \neq \beta
]

时，到底应该优先补贴用户，还是优先补贴商户。

其中：

[
\alpha
]

表示**商户规模对用户选择的影响**；

[
\beta
]

表示**用户规模对商户选择的影响**。

---

## 1. 最优补贴方向图分析

从“不同网络效应结构下的最优补贴方向”热力图看，区域主要分为三类：

* 蓝色：用户补贴最优；
* 绿色：商户补贴最优；
* 红色：双边补贴最优。

图中大部分区域为蓝色，说明在你当前参数设定下，**用户补贴在多数网络效应组合下具有更高净收益**。

但在右下方区域，也就是：

[
\alpha \text{ 较大},\quad \beta \text{ 较小}
]

商户补贴成为最优。这一区域对应的是：

> 用户非常看重商户规模，但商户对用户规模的敏感度相对较弱。

此时如果直接补贴用户，用户虽然被短期吸引过来，但由于商户侧吸引力不足，平台生态不够稳定；反而先补贴商户，更容易形成“商户多 (\rightarrow) 用户增长”的正反馈。

所以这张图可以得到一个很重要的结论：

[
\alpha>\beta \quad \Rightarrow \quad \text{商户补贴更有可能成为最优}
]

相反，当 (\beta) 较大时，图中多数区域为用户补贴最优。因为商户非常看重用户规模，平台只要先补贴用户，把用户规模拉起来，就可以带动商户自然流入。

因此：

[
\beta>\alpha \quad \Rightarrow \quad \text{用户补贴更有可能成为最优}
]

---

## 2. 不同网络效应结构下的最终份额比较

从最终份额柱状图可以看到：

### 双边弱网络效应

无补贴时平台 A 仍能保留一定市场份额：

[
x^*\approx 0.214
]

说明弱网络效应下，市场锁定程度不强，平台 A 不会立刻被挤出市场。

补贴后最终份额约为：

[
x^*\approx 0.795\sim0.799
]

但并没有达到接近 1 的完全锁定。这说明：

> 网络效应较弱时，补贴可以提升平台份额，但不一定造成赢家通吃。

---

### 对称强网络效应

无补贴时平台 A 几乎失败：

[
x^*\approx 0.0066
]

补贴后平台 A 接近锁定：

[
x^*\approx 0.993
]

说明强网络效应下，补贴的作用更明显，因为补贴一旦帮助平台跨过临界规模，就会触发双边正反馈。

---

### 用户更依赖商户

该情形为：

[
\alpha=1.6,\quad \beta=0.5
]

结果显示，商户补贴的净收益最高：

[
\text{商户补贴净收益}\approx 157.234
]

高于用户补贴的：

[
155.194
]

这符合理论预期。因为用户更看重商户规模，先补贴商户能更有效地提升用户侧吸引力。

---

### 商户更依赖用户

该情形为：

[
\alpha=0.5,\quad \beta=1.6
]

结果显示，用户补贴的净收益最高：

[
159.219
]

高于商户补贴的：

[
155.152
]

这也符合预期。因为商户更看重用户规模，用户补贴可以先扩大用户基础，再带动商户加入。

---

## 3. 拓展 1 总结

这一部分可以写成：

> 非对称网络效应会改变平台的最优补贴方向。当用户更依赖商户规模时，商户补贴更容易成为最优策略；当商户更依赖用户规模时，用户补贴更容易成为最优策略。双边补贴虽然更加稳健，但由于成本较高，只有在少数弱网络效应区域中表现更优。在当前参数设定下，用户补贴整体占优，但这并不是普遍规律，而是由用户补贴成本较低、初始条件和收益参数共同决定的。

# 二、拓展 2：最低有效补贴组合分析

这一部分研究：

> 平台 A 至少需要多大的补贴强度，才能达到锁定判据？

锁定判据为：

[
x^*\geq 0.8,\qquad y^*\geq 0.8
]

目标可以理解为：

[
\min (s_U+s_M)
]

其中：

[
s_U
]

是初始用户补贴强度，

[
s_M
]

是初始商户补贴强度。

---

## 1. 对称强网络效应

对称强网络效应下：

[
\alpha=\beta=1.2
]

最低有效补贴总强度为：

[
s_U+s_M=0.54
]

对应组合为：

[
s_U=0.45,\qquad s_M=0.09
]

这说明在对称强网络效应下，平台 A 需要一定强度补贴才能突破初始劣势。由于用户补贴在当前参数中成本收益表现较好，因此最低有效组合偏向用户侧补贴。

---

## 2. 用户更依赖商户

当：

[
\alpha=1.6,\quad \beta=0.5
]

最低有效补贴总强度为：

[
s_U+s_M=0.27
]

对应组合为：

[
s_U=0.03,\qquad s_M=0.24
]

这非常符合模型逻辑。

因为用户更依赖商户，所以平台 A 只需要重点补贴商户侧，就能迅速提升用户侧吸引力，进而触发正反馈。

也就是说：

[
\alpha \text{ 大} \Rightarrow \text{补贴商户的边际效果更强}
]

---

## 3. 商户更依赖用户

当：

[
\alpha=0.5,\quad \beta=1.6
]

最低有效补贴总强度同样为：

[
s_U+s_M=0.27
]

但对应组合变为：

[
s_U=0.27,\qquad s_M=0
]

这说明在商户高度依赖用户规模时，最有效的破局方式是先补贴用户，把用户规模做起来，然后吸引商户加入。

即：

[
\beta \text{ 大} \Rightarrow \text{补贴用户的边际效果更强}
]

---

## 4. 极强锁定市场

在极强锁定市场中：

[
\alpha=\beta=1.8
]

最低有效补贴总强度上升到：

[
s_U+s_M=0.84
]

对应组合为：

[
s_U=0.84,\qquad s_M=0
]

这说明网络效应越强，市场锁定越牢固，弱势平台要跨过临界规模所需的补贴强度越高。

可以总结为：

[
\alpha,\beta \uparrow
\quad \Rightarrow \quad
\text{最低有效补贴强度上升}
]

---

## 5. 拓展 2 总结

这一部分的结论很有建模价值：

| 情形      | 最低补贴强度 | 最优补贴方向  |
| ------- | -----: | ------- |
| 对称强网络效应 |   0.54 | 以用户补贴为主 |
| 用户更依赖商户 |   0.27 | 以商户补贴为主 |
| 商户更依赖用户 |   0.27 | 以用户补贴为主 |
| 极强锁定市场  |   0.84 | 更高强度补贴  |

可以写成：

> 最低有效补贴强度取决于网络效应结构。若某一侧网络效应更强，则补贴另一侧更容易触发正反馈。例如用户更依赖商户时，应优先补贴商户；商户更依赖用户时，应优先补贴用户。随着双边网络效应整体增强，市场锁定程度加深，平台 A 需要更高补贴才能突破锁定。

# 三、拓展 3：供给不足惩罚强度 (\rho) 的敏感性分析

这一部分分析：

[
\rho
]

对用户补贴、商户补贴、双边协调补贴效果的影响。

其中：

[
\rho
]

越大，说明供给不足对用户体验的惩罚越强。

---

## 1. 用户补贴对 (\rho) 非常敏感

从图中可以看到，当 (\rho) 较小时，用户补贴仍然有效，最终份额接近：

[
x^*\approx 0.994
]

但是当：

[
\rho \approx 7.67
]

之后，用户补贴策略突然失败，最终用户份额接近 0。

这说明用户补贴有明显的脆弱性：

> 当供给不足惩罚较弱时，用户补贴可以快速扩大需求并触发正反馈；但当供给不足惩罚足够强时，需求快速增长带来的体验损失会抵消补贴效果，导致平台无法完成反超。

这和现实很一致。

例如外卖平台大规模补贴用户，如果商户供给、骑手配送能力、履约系统没有跟上，用户虽然被优惠吸引来，但体验变差，很可能很快流失。

---

## 2. 最大供给不足随 (\rho) 增大反而下降

图中用户补贴的最大供给不足从约：

[
0.21
]

逐渐下降到：

[
0.075
]

表面上看，供给不足变小了。

但这并不表示用户补贴更稳定，而是因为 (\rho) 变大后，用户体验惩罚更强，用户份额增长被压制，甚至在高 (\rho) 下直接失败。因此：

[
\text{供给不足下降}
]

并不是供给改善，而是需求增长被抑制。

这一点报告中要解释清楚。

---

## 3. 商户补贴和双边协调补贴几乎不受 (\rho) 影响

商户补贴和双边协调补贴的最终份额基本稳定在：

[
x^*\approx 0.993
]

且最大供给不足始终接近 0。

原因是这两类策略都没有让用户需求单独快速膨胀，而是先增加供给侧，或者让两侧协调增长。

因此：

> 商户补贴和双边协调补贴对供给不足惩罚更稳健。

从净收益看：

* 用户补贴在低 (\rho) 下净收益最高，约为 (173)；
* 但高 (\rho) 下净收益骤降到约 (4)；
* 商户补贴基本稳定在约 (167)；
* 双边协调补贴基本稳定在约 (170)。

所以阶段 5 拓展 3 的核心结论是：

> 用户补贴是高收益但高风险策略；供给不足惩罚较弱时用户补贴最优，供给不足惩罚较强时商户补贴或双边协调补贴更稳健。

# 四、拓展 4：服务质量投资效率 (\mu) 与衰减率 (\delta)

这一部分分析服务质量投资机制是否稳健。

服务质量积累方程为：

[
\frac{dq}{dt}=\mu I-\delta q
]

其中：

* (\mu)：投资效率；
* (\delta)：服务质量衰减率。

---

## 1. 投资效率和衰减率对最终份额的影响

从最终份额热力图看，大部分区域为黄色，说明动态服务质量投资策略在多数参数组合下都能帮助平台 A 达到接近 1 的最终份额。

但是在左上角存在失败区域，对应：

[
\mu \text{ 较低},\quad \delta \text{ 较高}
]

也就是：

> 投资转化效率低，同时服务质量优势衰减快。

在这种情况下，平台即使持续投资，也难以形成足够稳定的质量优势，因此无法突破初始锁定。

例如当：

[
\mu=0.10,\quad \delta=0.10
]

时，最终用户份额只有：

[
x^*\approx 0.031
]

累计净收益为：

[
-62.469
]

说明投资几乎失效，且由于投入成本存在，反而造成亏损。

---

## 2. 投资效率越高，净收益越高

从净收益热力图看，随着 (\mu) 增大，累计净收益整体提高。

最高净收益出现在：

[
\mu=0.55,\quad \delta=0.01
]

此时：

[
x^*\approx 1.000
]

累计净收益约为：

[
181.191
]

这说明：

> 当投资效率高、质量优势衰减慢时，服务质量投资可以形成长期竞争壁垒，并带来较高净收益。

---

## 3. 衰减率越高，对低效率投资越不利

当 (\mu) 足够高时，即使 (\delta) 较大，平台仍然可以成功破局。

但当 (\mu) 较低时，(\delta) 的影响非常明显。

这说明服务质量投资是否有效，取决于二者相对关系：

[
\mu \text{ 要足够大，才能抵消 } \delta \text{ 的衰减}
]

如果平台技术投入、管理优化、配送体系建设等无法有效转化为用户和商户感知到的质量优势，那么服务质量投资就难以替代补贴。

---

# 六、阶段 5 总体结论

阶段 5 的结果可以总结为以下几点。

---

## 结论 1：补贴方向应匹配网络效应结构

当用户更依赖商户时：

[
\alpha>\beta
]

商户补贴更有效。

当商户更依赖用户时：

[
\beta>\alpha
]

用户补贴更有效。

这说明平台不能简单套用固定补贴策略，而应先判断市场中哪一侧对另一侧的吸引作用更强。

---

## 结论 2：最低有效补贴具有明显阈值

平台不需要无限补贴，而是需要找到跨过锁定门槛的最低有效补贴组合。

在当前实验中：

| 市场结构    | 最低有效补贴强度 |
| ------- | -------: |
| 用户更依赖商户 |     0.27 |
| 商户更依赖用户 |     0.27 |
| 对称强网络效应 |     0.54 |
| 极强锁定市场  |     0.84 |

说明市场锁定越强，平台越需要更高补贴才能破局。

---

## 结论 3：用户补贴对供给不足惩罚高度敏感

当供给不足惩罚较弱时，用户补贴收益最高。

但当：

[
\rho \gtrsim 7.67
]

用户补贴会突然失效。

这说明如果平台供给能力不足，单纯依靠用户补贴会带来较高风险。

---

## 结论 4：服务质量投资存在效率门槛

服务质量投资能否替代补贴，取决于：

[
\mu
]

和：

[
\delta
]

当投资效率低、衰减率高时，服务质量投资难以突破锁定，甚至可能导致亏损。

当投资效率高、衰减率低时，服务质量投资可以形成长期竞争优势。

---

## 结论 5：动态策略最适合作为稳健型长期策略

在服务质量投资中：

* 贪心策略收益最高，但风险较大；
* 长期策略稳定，但成本过高；
* 动态策略兼顾收益、稳定性和供给协调。

因此，若考虑长期运营稳定性，动态策略比单纯贪心策略更合理。

# 七、可以直接写进报告的阶段 5 分析

下面这段可以直接放入报告中：

阶段 5 对模型进行了参数敏感性与拓展分析。首先，在非对称网络效应下，补贴方向会随 (\alpha) 与 (\beta) 的相对大小发生变化。当 (\alpha>\beta) 时，用户对商户规模更加敏感，商户补贴更容易成为最优策略；当 (\beta>\alpha) 时，商户对用户规模更加敏感，用户补贴更容易成为最优策略。实验结果显示，在“用户更依赖商户”情形下，商户补贴的累计净收益最高；在“商户更依赖用户”情形下，用户补贴的累计净收益最高。这说明平台补贴策略应与双边网络效应结构相匹配。

其次，最低有效补贴组合分析表明，平台打破锁定存在明显补贴门槛。在“用户更依赖商户”和“商户更依赖用户”两种非对称情形下，最低有效补贴总强度均约为 (0.27)，但前者主要依赖商户补贴，后者主要依赖用户补贴；在对称强网络效应下，最低有效补贴强度上升至 (0.54)；在极强锁定市场中进一步上升至 (0.84)。这说明网络效应越强，市场锁定越牢固，弱势平台需要更高补贴才能跨越临界规模。

再次，供给不足惩罚强度 (\rho) 的敏感性分析显示，用户补贴对供给不足高度敏感。当 (\rho) 较小时，用户补贴可以维持较高最终份额和净收益；但当 (\rho) 增大到约 (7.67) 后，用户补贴策略突然失效，最终用户份额接近 0。相比之下，商户补贴和双边协调补贴几乎不受 (\rho) 影响，且供给不足始终接近 0。这说明单纯用户补贴虽然在供给约束较弱时收益较高，但在供给不足惩罚较强的市场中风险较大，平台应更加重视供给侧建设和双边协调。

最后，服务质量投资敏感性分析表明，投资效率 (\mu) 与服务质量衰减率 (\delta) 共同决定服务质量投资能否突破锁定。当 (\mu) 较低且 (\delta) 较高时，服务质量优势难以积累，平台 A 无法反超，甚至出现负净收益；当 (\mu) 较高且 (\delta) 较低时，平台 A 可以接近完全锁定市场，并获得较高净收益。进一步比较贪心策略、长期策略和动态策略可知，贪心策略在投资效率较高时净收益最高，但在低效率条件下容易失败并带来供给不足风险；长期策略最稳定但成本较高；动态策略能够在不同投资效率下保持较高份额和较好净收益，同时避免供给不足，是更稳健的长期运营策略。
