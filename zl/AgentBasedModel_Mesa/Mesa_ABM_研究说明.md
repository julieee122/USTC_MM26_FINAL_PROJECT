# Mesa 多智能体模型研究说明

## 1. 模型定位

前面阶段一和阶段二主要使用 ODE 动态模型研究双边平台竞争。ODE 模型刻画的是平均意义上的市场份额演化，例如：

\[
\frac{du}{dt}=\lambda_U(P_A^U-u),\qquad
\frac{dm}{dt}=\lambda_M(P_A^M-m)
\]

Mesa 多智能体模型则从个体层面出发，将用户和商户分别建模为 agent。每个用户和商户在每个离散时间步根据平台效用、网络效应、补贴、切换成本和随机偏好噪声选择平台 A 或 B。

因此，Mesa ABM 的主要作用不是替代 ODE 模型，而是作为个体层面的稳健性验证和机制扩展工具，用来回答：

> 当用户和商户存在个体差异、随机选择和离散互动时，ODE 模型得到的市场锁定、路径依赖和补贴效果是否仍然成立？

## 2. 当前 Mesa 模型包含的机制

当前基础版 ABM 已包含以下因素：

| 机制 | 模型实现 |
|---|---|
| 用户和商户两类主体 | `UserAgent` 和 `MerchantAgent` |
| 两个平台选择 | 每个 agent 在平台 A/B 中选择 |
| 双边网络效应 | 用户受商户份额影响，商户受用户份额影响 |
| 服务质量 | \(q_A^U,q_B^U,q_A^M,q_B^M\) |
| 补贴 | \(b_A^U,b_A^M,b_B^U,b_B^M\) |
| 用户价格和商户抽成 | \(p_A^U,p_B^U,r_A,r_B\) |
| 切换成本/惯性 | agent 留在原平台时获得惯性效用 |
| Logit 选择 | 根据效用差异以概率方式选择平台 |
| 个体偏好噪声 | \(\epsilon\sim N(0,\sigma^2)\) |

用户选择平台 A 的效用可理解为：

\[
V_i^U(A)=q_A^U+\alpha m_A+b_A^U-p_A^U+s_U I_i(A)+\epsilon_i^A
\]

商户选择平台 A 的效用可理解为：

\[
V_j^M(A)=q_A^M+\beta u_A+b_A^M-r_A+s_M I_j(A)+\epsilon_j^A
\]

其中 \(I_i(A)\) 表示该 agent 当前是否已经在平台 A 上，用于刻画切换成本或使用惯性。

## 3. Mesa ABM 可以研究的问题

### 3.1 ODE 结论的稳健性验证

研究问题：

> ODE 模型中观察到的市场锁定，在个体随机选择和有限样本下是否仍然出现？

可做实验：

- 固定与 ODE 相同的参数；
- 设置相同初始份额 \(u_A(0),m_A(0)\)；
- 多次运行 ABM；
- 比较 ABM 的平均轨迹与 ODE 的轨迹。

可观察指标：

\[
u_A(t),\quad m_A(t),\quad L_A(t)=\frac{u_A(t)+m_A(t)}{2}
\]

预期结论：

- 当网络效应较强时，ABM 仍会出现平台锁定；
- ABM 结果会围绕 ODE 结果波动；
- 个体随机性会使锁定过程出现一定方差。

### 3.2 个体偏好噪声对市场锁定的影响

研究问题：

> 用户和商户偏好越分散，市场越容易共存还是越容易锁定？

可扫描参数：

\[
\sigma_U,\sigma_M\in\{0,0.1,0.3,0.5,1.0\}
\]

其中 \(\sigma\) 越大，表示个体偏好随机性越强。

可观察指标：

- 最终平台 A 综合份额 \(L_A(\infty)\)；
- 市场集中度 \(C\)；
- 多次运行后的均值和标准差；
- 平台 A 锁定概率。

预期结论：

- 当 \(\sigma\) 较小时，agent 更容易跟随效用差异，网络效应更容易放大优势；
- 当 \(\sigma\) 较大时，随机偏好会削弱网络效应导致的集中，市场锁定可能变慢或不稳定；
- 偏好异质性可以作为平台共存的一种缓冲机制。

### 3.3 有限市场规模下的随机锁定

研究问题：

> 在有限数量用户和商户下，即使初始条件相同，市场最终是否可能锁定到不同平台？

可扫描参数：

\[
N_U,N_M\in\{100,500,1000,5000\}
\]

可做实验：

- 对每个市场规模重复运行 30 到 100 次；
- 统计平台 A 锁定、平台 B 锁定、共存的比例。

可观察指标：

- \(P(\text{平台A锁定})\)；
- \(P(\text{平台B锁定})\)；
- \(L_A(\infty)\) 的分布；
- 不同市场规模下结果方差。

预期结论：

- 市场规模较小时，随机波动更大，锁定方向可能更不稳定；
- 市场规模较大时，ABM 平均结果更接近 ODE 模型。

### 3.4 补贴策略在个体层面的效果

研究问题：

> 补贴是否能在个体随机选择下帮助弱势平台跨过临界规模？

可做实验：

- 平台 A 初始落后，例如 \(u_A(0)=m_A(0)=0.3\)；
- 扫描统一补贴 \(b\)；
- 比较无补贴、低补贴、中等补贴和高补贴。

可观察指标：

- \(L_A(t)\)；
- 平台 A 锁定概率；
- 达到 \(L_A>0.6\) 所需时间；
- 多次运行下的平均路径和置信区间。

预期结论：

- 补贴在 ABM 中也可能表现出临界门槛；
- 在门槛附近，不同随机种子可能导致不同结果；
- 这可以解释现实平台竞争中“同样补贴策略不一定每次都成功”的现象。

### 3.5 ODE 与 ABM 的对比

研究问题：

> 平均场 ODE 模型和个体 ABM 模型的结论是否一致？

可做实验：

- 使用同一组参数分别运行 ODE 和 ABM；
- ABM 重复运行多次；
- 将 ODE 曲线与 ABM 均值曲线画在同一张图中。

可观察指标：

- ODE 的 \(u_A(t),m_A(t)\)；
- ABM 多次仿真的均值；
- ABM 多次仿真的标准差区间。

预期结论：

- ODE 描述平均趋势；
- ABM 展示个体随机性带来的波动；
- 如果两者总体趋势一致，可以增强模型结论的可信度。

## 6. 论文中可以强调的价值

Mesa ABM 的价值主要体现在三个方面：

1. **个体异质性**：ODE 模型使用平均份额，ABM 能刻画不同用户和商户的随机偏好。
2. **随机稳健性**：ABM 可以通过多次随机仿真检验市场锁定结论是否稳定。
3. **机制解释**：ABM 可以展示平台锁定不是由单个代表性主体决定，而是由大量个体选择在网络效应下共同累积形成。

因此，ABM 部分可以作为全文的稳健性验证和 AI/多智能体特色补充。
